# Scripts

Utility scripts for project setup and management.

## Available Scripts

### quick_start.py
Quick setup verification script. Checks dependencies, configuration, and creates necessary directories.

**Usage:**
```bash
python scripts/quick_start.py
```

### setup_cloud.py
Interactive cloud setup wizard. Guides you through API key configuration and cloud service setup.

**Usage:**
```bash
python scripts/setup_cloud.py
```

### run_dashboard.py
Launcher script for the Streamlit dashboard.

**Usage:**
```bash
python scripts/run_dashboard.py
```

Or use Streamlit directly:
```bash
streamlit run app/dashboard.py
```
